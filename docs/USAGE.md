# Utilisation

## Raccourcis
- **ALT** : déplacer l’icône (si option activée)
- **Clic droit sur l’icône** : options citeturn15search0

## Fonctions principales (UI)
- **Fenêtre principale** : liste de vos téléports / compétences de voyage
- **Drag & drop** : réorganisation des raccourcis citeturn15search0
- **Options** : réglages d’affichage / comportement (via clic droit) citeturn15search0

## Organisation & astuces
- Placez vos téléports “les plus utilisés” en haut.
- Regroupez par zones/expansions (ex: Gondor, Mordor, Umbar) si votre version contient ces cartes.
- Pensez à sauvegarder via la fenêtre d’options (si votre version a un bouton Save).

## Données & synchro
- Les réglages et positions sont stockés par personnage :
  `Documents\The Lord of the Rings Online\PluginData\<Compte>\<Serveur>\<Perso>\`
- En cas de bug après une mise à jour : supprimez uniquement le fichier de settings du plugin, relancez, reconfigurez.
